nouncaptcha

(2014)

nouncaptcha's website. (featuring a working example captcha)

Nouncaptcha was created for my undergraduate honor's thesis requirement. It was an effort to create an alternative to text based CAPTCHAs, such as reCAPTCHA, and was made with modern internet-connected devices in mind.

The CAPTCHA was designed to be easy to use with either a keyboard and mouse, or with just a touchscreen. Creating a theoretically secure CAPTCHA with such design constraints proved to be an enthralling challenge!

Rebalance Calc

(2016)

Rebalance Calc's website.

Rebalance Calc is a tool to distribute a desired asset allocation among taxable and tax-advantaged accounts. It is designed to optimize for tax efficiency. I originally built a similar tool using excel, but found myself missing the ability to program my solution in a familiar language. I also wanted the ability to separate data from presentation (my spreadsheet had quickly became a confusing mess with cells containing intermediate state).

After some thought, I felt a simple web app could address my needs, and potentially the needs of many others. I built the site, and have found it quite useful for my fairly novice investing use case.

Smart Sync

(2015)

I created Smart Sync after getting frustrated waiting for the built-in Windows 10 mail app to sync changes to the server. After some investigation, I discovered "triage" events (eg: marking as read, moving to a folder, etc) flush to the server on a schedule, and do not propagate immediately. Being lazy at heart, I refuse to press the "refresh" button to sync my work before switching to a web mail interface or my phone.

After some investigation, I discovered a nifty API to access email mailboxes on my machine, listen for change events, and automatically invoke a sync. I consumed the API using a Windows 10 universal app, as the ApplicationModel.Email namespace is only available to UWP apps. The solution I created can launch the mail app, and immediately syncs local changes to the server.

Now I can rest assured knowing my email is synced when I switch to my phone or a different computer.

Dev Box Setup Scripts

(2016)

I generally work on small projects in Ubuntu VMs, and have found it tedious to get these machines set up exactly how I like by hand. This task was a perfect candidate for automation, so I created a series of scripts and pushed them to GitHub. Now it's quick and easy to install the software packages I rely on, to set up git and GitHub public key auth, and to properly configure an SSH server (it's as simple as ctrl-c & ctrl-v).

These scripts simplify my life by making it easy to get an Ubuntu VM to a known good state. They also guarantee my production and development environments have similar, if not identical (where appropriate), configurations!
