Rebalance Calc

started in 2016

RebalanceCalc determines what accounts should hold what investments, and specifically optimizes for tax efficiency.

I originally built a similar tool using excel, but found the spreadsheet difficult to manage and maintain.

After some thought, I felt a simple web app could address my needs, and potentially the needs of many others. I built the site, and have found it very useful while rebalancing my investments.

Nouncaptcha

started in 2014

Nouncaptcha was created for my undergraduate honors thesis. It was an effort to create an alternative to text based CAPTCHAs, such as reCAPTCHA, and was made with mobile devices in mind.

Nouncaptcha's design makes it easy to use with either a keyboard and mouse, or with just a touchscreen. Creating a theoretically secure CAPTCHA with such a widely accessible design proved to be an enthralling challenge!

Dev Box Setup Scripts

started in 2016

I've provisioned a number of Ubuntu (and other Debian-based) machines, VMs, and WSL instances over the years. As a consequence, I've spent a lot of time getting my dev environments set up.

In an effort to simplify things, I created a series of configuration scripts and pushed them to GitHub. Now it's quick and easy to install the software packages I rely on, set-up git with GitHub public key authentication, and to properly configure an SSH server.
